This Site

So, this site would have to be considered the most recent project, even though it was set up to contain all of the other projects that will be created throughout this class. This site has the projects grouped by their Rotation, which are accessed though the "Projects" dropdown menu. Inside each "Rotation" page, there is a list of the projects that were in that rotation. Each one when clicked will show a description of the project, any additional information about it, and pictures if possible.

There will eventually be an "About Me" page up, and the schedule on the sidebar will be updated whenever new projects are added. This "Most Recent Project" section will also be updated with every new project.
